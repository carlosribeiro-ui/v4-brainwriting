# V4 Brainwriting + Supabase

## Variável de ambiente no Netlify

- `DATABASE_URL`

## Setup do banco

1. Abra o SQL Editor do Supabase.
2. Rode o arquivo `schema.sql`.

## Deploy no Netlify

1. Suba esta pasta por Git para o Netlify.
2. Configure `DATABASE_URL`.
3. Faça o deploy.

## Rotas

- `POST /api/vote`
- `GET /api/results`
